Xin chào! Đây là mã nguồn website Portfolio cá nhân của **Shio (Dương Đức Cương)** — một Junior Developer & học sinh THPT Chuyên Trần Phú, Hải Phòng. Website được thiết kế theo phong cách **Kawaii Cute + 3D Modern** vô cùng độc đáo và sinh động! ✨

## 🌟 Tính năng nổi bật (Features)

*   **🎨 Giao diện Kawaii & Hiện đại:** Thiết kế thẻ bo góc (Glassmorphism), màu sắc Pastel ngọt ngào (Hồng, Tím, Xanh ngọc).
*   **🌐 Đa ngôn ngữ (Bilingual):** Hỗ trợ chuyển đổi mượt mà giữa Tiếng Việt (VI) và Tiếng Anh (EN).
*   **🌓 Chế độ Sáng/Tối (Dark/Light Mode):** Tự động điều chỉnh giao diện thông minh mà không làm vỡ màu ảnh tĩnh hay ảnh GIF.
*   **🎵 Background Music:** Trình phát nhạc lofi chill chill tự động lặp (Auto loop).
*   **🧊 Hiệu ứng 3D (Three.js):** Cảnh nền 3D tương tác với các hành tinh và các vì sao.
*   **✨ Hoạt ảnh mượt mà (Framer Motion):** Các hiệu ứng hover, scroll reveal mượt mà chuẩn web xịn.
*   **🎮 Gaming Hub:** Khu vực hiển thị thông tin game cực chất.

## 🛠️ Công nghệ sử dụng (Tech Stack)

*   **Framework:** Next.js 15 (App Router)
*   **Ngôn ngữ:** TypeScript
*   **Style:** Tailwind CSS, Shadcn UI
*   **Animation:** Framer Motion
*   **3D Graphics:** React Three Fiber, Drei
*   **Icons:** Lucide React

## 🚀 Hướng dẫn cài đặt (Getting Started)

Làm theo các bước sau để chạy dự án này trên máy của bạn:

**1. Clone dự án về máy**
```bash
git clone https://github.com/ShioVn/shio-portfolio.git
cd shio-portfolio
```

**2. Cài đặt thư viện**
```bash
npm install
# hoặc yarn install
```

**3. Khởi chạy môi trường phát triển (Development Server)**
```bash
npm run dev
# hoặc yarn dev
```

Mở trình duyệt và truy cập [http://localhost:3000](http://localhost:3000) để xem kết quả nhé! 🎉

## ✏️ Cách chỉnh sửa nội dung (Customization)

Toàn bộ nội dung của trang web được quản lý tập trung tại **2 file duy nhất** để bạn dễ dàng tuỳ biến mà không cần đụng vào UI Components:
*   `src/config/constants.ts`: Quản lý thông tin cá nhân, mạng xã hội, dự án (projects), link hình ảnh/GIFs, và kỹ năng (skills).
*   `src/config/i18n.ts`: Quản lý toàn bộ text (văn bản) cho cả Tiếng Việt và Tiếng Anh.
